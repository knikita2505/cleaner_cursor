# [SCREEN] secret_album

## Назначение
Управление скрытыми фото и видео.

---

## UI

- Галерея скрытых элементов (grid).
- Кнопка `Add from Photos`:
  - открывает стандартный пикер;
  - выбранные фото копируются/перемещаются в Secret Album (в зависимости от политики).
- Кнопка `Move back to Photos` для выбранных элементов.

---

## Логика

- Для скрытия:
  - медиа файлы переносим в секретный альбом, оригиналы предлагаем удалить;
  - фото / виде переносятся без потери качества
- Для возврата:
  - для добавленных файлов должна быть кнопка "Поделиться", которая вызывает стандратное окно поделиться фото, где можно сохранить файл обратно в галерею или поделиться в других приложениях

---

## Аналитика
