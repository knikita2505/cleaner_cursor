# [SCREEN] system_tips_list

## Назначение
Экран со списком системных советов (storage, battery, performance).  
Используется для мягкого удержания и формирования ощущения «умного» ассистента.

---

## UI

(как мы уже описывали ранее — список секций Storage / Battery / Performance, каждая с карточками советов и раскрытием подробностей, CTA перехода к экранам очистки/battery.)

---

## Логика
- Советы ранжируются по актуальности:
  - если storage >85% — storage tips наверху;
  - если батарея часто низкая — battery tips выше.
- Советы можно обновить по кнопке `Refresh`.
