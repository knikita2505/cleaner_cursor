# [SCREEN] paywall_main

## Назначение
Основной paywall с триалом и подписками. Может иметь варианты A/B (визуал и тексты).

---

## Основной layout

### Верхняя часть
- Иконка приложения на soft-background.
- Заголовок: `Clean up your iPhone fast & easy`
- Подзаголовок: список ключевых выгод:  
  - удаление дубликатов  
  - сжатие видео  
  - секретный альбом  
  - советы по батарее и устройству

### Блок триала
- Тумблер `Free Trial Enabled / Disabled`  
  - При включении: текст `7 days free, then X.XX/week`.
  - При выключении: показываем прямую оплату без триала.

### Блок тарифов
Две опции:
1. Годовой план
   - `Most popular`
   - Цена: например `€34.99/year` + пересчёт на неделю.
2. Недельный план
   - Цена: `€6.99/week`.

Выбор — radio-стиль.

### Текст выгод
Небольшой блок bullet-поинтов: без рекламы, полный доступ к функциям, приоритет в новых фичах.

### Кнопки
- Основная: `Try for Free` (или `Continue`, если триал выключен).
- Вторичная: `Skip for now` (в правом верхнем углу или внизу небольшим текстом).

---

## Логика

### Поведение триала
- При нажатии `Try for Free`:
  - инициируем покупку через StoreKit;
  - в случае успеха — устанавливаем статус Premium, записываем в аналитике;
  - в случае отказа — показываем ненавязчивый баннер.

### A/B-тесты
- Тексты, порядок тарифов, наличие лайфтайма и копирайты могут меняться через `feature_flags` + `subscriptions_and_free_limits`.

### Принудительный показ
Paywall может показываться:
- из онбординга;
- при достижении free-лимита;
- при нажатии `Start Free Trial` на дашборде;
- вручную из `feature_flags`.
