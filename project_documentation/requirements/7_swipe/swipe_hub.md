# [SCREEN] swipe_photos_hub

## Назначение
Хаб-экран режима **Swipe Clean** — геймифицированный просмотр фото с решением «оставить/удалить».

---

## Структура

- Заголовок: `Swipe photos`
- Summary:
  - `0% sorted` / прогресс-бар;
  - `Months in queue: <N>` (например, октябрь, сентябрь и т. д.).
- Список карточек по месяцам:
  - `October 2025 — 0/618 reviewed`
  - `September 2025 — 100/516 reviewed`…

Тап по месяцу → `swipe_session` для этого диапазона.

---

## Логика
- Прогресс по каждому месяцу хранится локально.
- Swipe-режим работает только при наличии достаточного количества фото.

---

## Аналитика
- `swipe_hub_open`
- `swipe_month_open`
