# [SCREEN] swipe_session

## Назначение
Экран Tinder-образного свайпа фотографий: вправо — сохранить, влево — удалить.

---

## UI

- Full-screen фото.
- Снизу:
  - счетчик `Photo X of Y`.
  - две крупные кнопки:
    - `Keep`
    - `Delete`
- Свайпы:
  - влево = Delete photo
  - вправо = Keep photo
- Опционально — иконки для быстрого действия.

---

## Логика

- Фото подаются по выбранному месяцу.
- Решения накапливаются в буфере.
- При нажатии `Apply` (кнопка в правом верхнем углу):
  - выполняются реальные удаления.
- Можно использовать soft-undo (последний свайп откатить).

---

## Аналитика
- `swipe_session_start`
- `swipe_decision` (action: keep/delete)
- `swipe_apply`
