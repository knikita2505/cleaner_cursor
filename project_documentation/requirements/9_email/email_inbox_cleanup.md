# [SCREEN] email_cleaner_inbox_cleanup

## Назначение
Упрощённая работа с рассылками: отписка/удаление пачками.

---

## UI

- Список отправителей (по домену/адресу):
  - имя/бренд;
  - примерный тип (promo/social/updates);
  - количество писем (`999+`, `43` и т. д.).
- Для каждого отправителя:
  - чекбокс или пара кнопок `Keep` / `Unsubscribe+Clean`.
- Кнопка `Apply` внизу:
  - `Clean X senders • Y emails`.

---

## Логика

- Для выбранных отправителей:
  - если поддерживается API — выполняем отписку;
  - удаляем существующие письма из папки Inbox/Promotions;
- Важно: всё это нужно реализовать щадяще, без "опасных" действий!

---

## Аналитика
- `email_cleanup_open`
- `email_sender_action` (keep/cleanup)
- `email_cleanup_success`
