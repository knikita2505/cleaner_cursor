## CONTACTS-001 — Анализ контактов
- Находить:
  - дубликаты,
  - пустые контакты,
  - формат, требующий нормализации.
- Логировать количество проблем.

## CONTACTS-002 — Merge / Delete
- Merge:
  - перенос всех номеров/почт в один контакт.
- Delete:
  - удаление пустых контактов.
- Ошибки:
  - отсутствие прав.
