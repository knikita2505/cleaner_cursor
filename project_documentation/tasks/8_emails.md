## EMAIL-002 — Анализ почты
- Загрузить список последних писем (метаданные).
- Группировать отправителей.
- Определить «спамность».
- Логировать:
  - количество спам-отправителей,
  - количество писем.

## EMAIL-003 — Delete / Unsubscribe
- Mass delete писем по отправителю.
- Unsubscribe через WebView.
- Логировать:
  - `email_delete_all`
  - `email_unsubscribe`
