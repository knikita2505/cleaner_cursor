# battery_animation

## Назначение
Экран с псевдо-анимацией зарядки — визуальная фича, отличающая приложение от конкурентов.  
Создаёт эффект «зарядного буста» с плавной aurora-анимацией.

---

## Основной сценарий
1. Пользователь открывает экран из Battery Dashboard.
2. Видит красивую градиентную анимацию, зависящую от уровня заряда.
3. Может включить звук (опционально).
4. Может перейти в советы или назад.

---

## Layout и UI

### Fullscreen Animation
Экран полностью покрыт:
- animated Aurora gradient (3–4 слоя),
- soft glow волн,
- реактивные частицы (опционально, low-intensity).

Анимация отвечает текущему уровню батареи:
- <20% → красноватые волны, быстрые пульсации.
- 20–80% → нормальная фиолетово-голубая волна.
- >80% → спокойные медленные переливы.

---

### Center Battery Indicator
В центре большой компонент:

- Circular ring 160–200pt  
  - анимация заполнения кольца,
  - прогресс = battery %.
- Внутри:
  - `72%` — большой Bold текст.
  - статус: `Charging`, `Discharging`.

Анимация кольца синхронизирована с фоном:
- при зарядке: пульсирует,
- при разрядке: медленная волна.

---

### Info Chips (under the circle)
Маленькие капсулки  — 14–15pt текст:
- `Temperature: Normal`
- `Condition: Good`
- `Current drain: Medium`

---

### Buttons
- Primary: **Back to Dashboard**
- Secondary: **View Battery Tips**

---

## Логика
- Анимация реагирует на реальное изменение battery level.
- При charging state кольцо пульсирует каждые 2 секунды.
- Можно ставить `isLowPowerModeEnabled` → меняет фон.

---

## Edge Cases
- На старых устройствах — fallback анимация (менее ресурсоёмкая).
- Если батарея не отслеживается → показать placeholder.
