# onboarding_01_welcome

## Назначение
Первый экран онбординга. Нужен для презентации продукта, формирования доверия и создания премиального визуального впечатления.

## Элементы
- Фон: Aurora gradient (#2F3DAF → #6B3BDB → #8B5CFF)
- Иллюстрация 220pt ( тема: очистка / чистота / устройство )
- Заголовок: "Cleaner for iPhone"
- Подзаголовок: "Free up space. Organize your photos. Keep your device fast."
- Прогресс-индикатор (1/4)
- Primary CTA: "Continue"
- Secondary CTA: "Skip" (opacity 40%)

## Логика
- Нажатие Continue → следующий экран онбординга
- Skip → переход на paywall_intro

## Анимации
- Fade-in и лёгкий scale-in иллюстрации (0.35s)
- Slide-in текста (0.25s)
